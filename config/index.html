# 配置相关

  • 配置使用的 json5，兼容 json 且更加灵活，可以支持 注释。
  • 务必使用 https://www.lddgo.net/string/json5open in new window 校验 json5 格式。
  • 压缩配置地址（选择 gzip 压缩）：https://www.baidufe.com/fehelper/en-decode/open in new window

# 配置文件路径

  • config/config.json
  • 与 index.js 同目录/config/config.json
  • index.js 同级 config.json
  • 以上配置后缀改为 .json5 同样有效
  • 当然你可以不用文件配置，直接使用环境变量 BILITOOLS_CONFIG（配置按照要求 gzip 压缩open in new window）

# 配置介绍（开发查看）

所有配置都登记在 types/config.tsopen in new window 文件中

解释：
  • number: 数字 例如：123。
  • string: 字符串 例如："ojbk"。
  • boolean: 布尔值 true 或者 false。
  • []: 数组 例如：number[] 具体可以是 [1,2,3]。
  • targetLevel?: number; ? 表示 targetLevel 是可选的配置。
